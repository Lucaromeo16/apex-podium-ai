# ApexPodium AI

Simple frontend prototype for an F1 podium prediction dashboard.

## Files included

- `package.json`
- `index.html`
- `src/main.jsx`
- `src/App.jsx`
- `src/styles.css`

## Run locally

```bash
npm install
npm run dev
```

Then open the local URL printed in the terminal (typically `http://localhost:5173`).

## Notes

- Driver predictions in `src/App.jsx` are mock data.
- UI is intentionally simple for easy iteration.
